# Yunzai-Bot Docs

基于[vitepress](https://github.com/vuejs/vitepress)构建

## 参与贡献

本仓库为Yunzai-Bot文档仓库，欢迎大家参与贡献。
***环境要求: nodejs v18以上版本, pnpm v9***

1. 拉取本仓库
```
git clone https://github.com/DenFengLai/Yunzai-Docs
```

2. 进入对应的目录并切换分支
```
cd Yunzai-Docs
git checkout dev
```

3. 安装依赖
```
pnpm install
```

4. 运行
```
pnpm dev
```

5. 提交修改

```
git add .
git commit -m "xxx"
git push
```
