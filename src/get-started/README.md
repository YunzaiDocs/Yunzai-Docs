::: tip 推荐使用脚本安装
非常推荐Linux用户使用 [TRSS Script](https://trss.me/) 安装项目，环境部署维护一条龙，全程自动无需手动安装环境以及项目插件，还能方便快捷的控制自启动、多窗口、以及自动更新项目及插件，管理起来超方便（TRSS打钱）
Windows用户也可以用，体验也是相当不错的
:::
