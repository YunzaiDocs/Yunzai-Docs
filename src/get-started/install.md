## 选择合适的安装位置

在你的设备上选择一个合适的位置并打开终端，我们通常建议你选择一个容易查找或移动的文件夹作为父目录，目录的命名最好不要包含特殊字符，如：中文、空格、斜杠与引号等。

建议使用纯英文路径，如：`~/Yz/Yunzai-Bot` 或 `D:/Yz/Yunzai-Bot`。

::: tip Windows打开终端
使用文件管理器打开对应文件夹，在空白处右键选择`在终端中打开`(Win11)或在上方的地址栏中输入 `cmd` 并按下回车。
:::
